## Useful Software

This is a list of software I first install:

```
Pamac => Package Manager (enable AUR)

$ sudo pacman-mirrors --fasttrack && sudo pacman -Syyu
Enable firewall => sudo ufw enable

Vivaldi with ffmpeg
Visual Studio Code
Timeshift (daily)
Steam
Transmission
LibreOffice
Genymotion (I'm a mobile dev)
```
